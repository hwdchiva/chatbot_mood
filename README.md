# Chatbot Mood
Chatbot with multiple moods/personalities, leveraging OpenAI prompt interface.

## Environment

- Python 3.16.3
- Environment variable 'OPENAI_API_KEY' must be set to your OpenAI key starting with 'sk-proj-...'
